Purpose: It acts as a marketplace comparison tool where users can evaluate different suppliers based on metrics like price, distance, rating, and available stock before making a purchase.
Header & Navigation: Includes a sticky navigation bar with links to the Home, Dashboard, My Orders, and Profile pages, along with a "Logout" button. It also features a custom dropdown to switch languages (English, Hindi, Bengali) using Google Translate.
Animated Hero Section: The top section features a playful 3D animated floating cube showing various vegetable emojis (🍅, 🥕, 🧅, etc.) to visually represent the platform's focus on fresh ingredients.
Interactive Comparison Cards: The main content consists of a grid of 3D "flip cards" representing different suppliers (Fresh Mart, Local Farm, and Spice Hub).
Card Front: Displays quick stats like price per kg, star rating, distance in kilometers, and available stock.
Card Back: Reveals more details about the supplier's practices (e.g., "Organic farming practices") and includes an "Order Now" button that links the user to order.html.
Scripts: The bottom of the file contains JavaScript that handles the Google Translate integration. It uses localStorage and cookies to remember the user's selected language and seamlessly translate the page on load without showing the default Google Translate banner.
